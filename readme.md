# CNode Lighten
CNode社区非官方小程序客户端，现已实现基本功能。

感谢开源项目[towxml](https://github.com/sbfkcel/towxml)提供小程序的Markdown渲染功能。

发主题的功能故意没做，因为感觉移动端发主题十分难用，用处不多。但以后可能会做。
